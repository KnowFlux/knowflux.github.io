# Current Strengths
* Consistent font styles for headings vs. body
* Clear layouts on homepage (left/right column)
* Consistent two colors (KnowFlux <span style = "color:#4392F1">blue</span> & KnowFlux <span style = "color:#F46036">orange</span>)
* Smooth transforms of links
* Underlines
* Main spacing works

# Weaknesses: Things That Feel Messy or Unfinished
* Long lists of cards - take up too much space, feel unorganized
* Navigation menus - too many submenus
* Logo is right on top of banner
* Book text feels cramped on mobile layout - not enough breathing room with so many elements
* Inconsistent buttons - homepage/main pages border radius; reading pages no border radius, plus shadow and press effect
* Broken links
* Main pages: black headers; all subsidiary heades: blue, different size
* Subheaders on subsidiary pages entirely absent

# Visitor Vibe
KnowFlux should feel like a vibrant, literary dimension where words almost come to life. Moving through the site feels like navigating a new universe, where attractive aesthetics create a poetic, mystical, but life-filled feel. Soft, slow, breathing animations instantly spark curiosity and interest. Finally, crisp typography and logical, clean-looking layouts tie the experience together.

# New Design Principles
* Main Colors: KnowFlux Blue - #4392F1 & Orange - #F46036
* Accent Color: #ff3c00
* Heading font: Anton, cursive
* Body font: Open Sans, sans-serif
* Tiny spacer: 10px
* Small spacer: 20px
* Medium spacer: 40px
* Large spacer: 50px
* Extra large spacer: 100px

# Color Scheme
* C2A (call to action, <div id = "action">): #4392F1
* C2A Transform: #ff3c00
* Card Main: #4392F1
* Card Transform: #F46036
* Homepage Text Link Main: #000
* Homepage Hover: #4392f1

# New Spacing Principles
* Consistent PER-PAGE spacing - if page uses one <div class = "spacer v10"> the rest of the page must employ the same spacer
* Make spacer heights more consistent and pattern like - not every spacer from 0 - 50 and then a 100px


# Reflection
* I want the KnowFlux mood to feel vibrant and lifelike, but also unurried, with a slow, breathing feel to animations. 
* The first couple of pages I will make major improvements to are all the pages and chapters sections, which feel overly cluttered and in some ways ugly in both mobile and desktop layouts. 
* The biggest visual problem is undoubtedly the navigation menus, which are overly large and can clutter the screen as there are three full layers to them.
